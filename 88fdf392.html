Linux|命令集合

2022-01-19 Linux 305 words
Linux

系统命令

su ： 仅切换身份。
su - ： 身份与$PATH工作路径都切换。
su 和 su- 最大的本质区别就是：前者只是切换了root身份，但Shell环境仍然是普通用户的Shell；而后者连用户和Shell环境一起切换成root身份了。（可通过pwd 和 $PATH 查看区别）

Docker

// 查看 docker 守护进程运行状态  
service docker status  

// 启动/关闭/重启 docker 守护进程
service docker start/stop/restart  

// 返回所有容器和镜像的数量、docker使用的执行程序和存储驱动，以及docker的基本配置信息  
docker info  

// 显示正在运行的容器  
docker ps  
docker container ls  

// 显示已安装的容器
docker ps -a

// 查看某容器所有状态信息(ip...)
docker inspect [docker_name 或 docker_id]

// 在已经启动的容器内部运行进程  
docker exec -it [docker_name 或 docker_id] /bin/bash //创建tty终端，捕捉STDIN，在容器内启动一个交互式命令  
docker exec -d docker_name/docker_id touch /home/test.txt //-d表示后台进程，在指定容器中运行一个后台进程'touch /home/test.txt'

Java

// 切换java版本  
update-alternatives --config java

// 设置java版本优先级  
update-alternatives --install /usr/bin/java java /usr/local/java/jdk1.8.0_311/bin/java 1110
