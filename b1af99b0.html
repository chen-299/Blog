2.getshell

/css 和 /manual 没有发现什么敏感信息，/index.html 是 img 4张图片的展示界面。
/js 下存在 main.js 访问得到 AES密钥 SecretPassphraseMomentum ：

function viewDetails(str) {

 window.location.href = "opus-details.php?id="+str;
}

/*
var CryptoJS = require("crypto-js");
var decrypted = CryptoJS.AES.decrypt(encrypted, "SecretPassphraseMomentum");
console.log(decrypted.toString(CryptoJS.enc.Utf8));
*/

回到 index.html 点击图片，url重定向到 opus-details.php?id= +str。
使用 /opus-details.php?id=<img src=1 onerror=alert(document.cookie)> 获取 Cookie:U2FsdGVkX193yTOKOucUbHeDp1Wxd5r7YkoM8daRtj0rjABqGuQ6Mx28N1VbBSZt 。
将得到的 Cookie 作为密文，与 AES密钥 进行AES解密，得到字符串 auxerre-alienum## 。
80端口没有找可以登录的入口，故将解密得到的字符串拆解为用户名和密码字典：
┌──(root㉿kali)-[~/Desktop]
└─# cat db.txt

auxerre-alienum
auxerre-
auxerre
-alienum##
alienum##
alienum
