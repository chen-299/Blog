符号

'xxx'::<type> ： 将xxx转化为指定type常量。
<xxx>as<yyy> ： 以yyy的结果定义xxx。
$ ： 引用常量。
$$ ： 包裹函数主体。
volatile ： volatile指函数可以修改数据库, 函数参数值相同的情况下, 可以返回不同的结果, 所以volatile函数在执行过程中优化器对它的处理是每一行都需要执行一次volatile函数。

OracleDB(:1521)

报错

Q1.远程连接数据库提示未监听：
ORA-12541:TNS:no listener
A1.未启动数据库监听：
[oracle@localhost ~]$ lsnrctl status //查询
[oracle@localhost ~]$ lsnrctl start //启动
Q2.远程连接数据库提示不知名服务
ORA-12514: TNS:listener does not currently know of service requested in connect descriptor
A2.监听启动了但数据库未启动
[oracle@localhost ~]$ sqlplus / as sysdba
SQL> startup
ORACLE instance started.
Q3.登录提示连接到空闲实例：
SQL> conn / as sysdba
Connected to an idle instance.
A3.未启动数据库：
SQL> startup
ORACLE instance started. //启动

登录OracleDB

[oracle@localhost ~]$ sqlplus / as sysdba;
**=**
[oracle@localhost ~]$ sqlplus /nolog  //连接数据库暂不登录
SQL> conn / as sysdba;

查询某角色具有的权限（以DBA为例）

select * from role_tab_privs where role = 'DBA'; --查询角色DBA具有的对象权限（对象权限赋予角色）
select * from role_sys_privs where role = 'DBA'; --查询角色DBA具有的系统权限（系统权限赋予角色）
select * from role_role_privs where role = 'DBA'; --查询角色DBA具有的角色（角色赋予角色）

查询用户账户具有的权限（以SYS为例）

select * from DBA_tab_privs where GRANTEE = 'SYS';
select * from DBA_role_privs where GRANTEE = 'DBA';
select * from DBA_sys_privs where GRANTEE = 'DBA';

ROLE_TAB_PRIVS

ROLE_TAB_PRIVS describes table privileges granted to roles. Information is provided only about roles to which the user has access.
