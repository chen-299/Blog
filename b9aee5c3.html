DVWA|CommandInjection

2021-12-17 渗透测试 814 words
DVWA CommandInjection

0.漏洞原理

在操作中系统中 &、&&、|、|| 都可以作为命令连接符使用，用户通过浏览器提交执行命令，由于服务端没有针对执行函数做过滤，导致在没有指定绝对路径的情况下就执行命令。
&&和&的区别在于，&&是执行完前面的命令然后执行后面的命令，&是不管前面的命令是否值执行，后面的都执行。

1.Low_level

1.1 尝试注入

在ip地址后输入&ipconfig：

1.2 源代码

<?php 
if(isset($_POST['Submit'])) {
   // Step1 代码接收用户输入的ip  
   $target = $_REQUEST['ip'];   
   // Step2 根据操作系统的区别，对接收ip执行命令不同的ping操作
   if(stristr(php_uname('s'),'Windows NT')){
       //Windows
       $cmd = shell_exec('ping'.$target);
   }
   else{
       //*nix
       $cmd = shell_exec('ping -c 4'.$target);
   }
   // Final 反馈结果没有进行任何过滤或限制
   $html .= "<pre>{$cmd}</pre>";
}
?>

2.Medium_level

2.1 尝试注入

在ip地址后输入&ipconfig：

2.2 源代码

<?php
if (isset($_POST['Submit'])){
   // Step1 代码接收用户输入的ip  
   $target = $_REQUEST['ip'];
   // Step2 设置过滤列表  
   $subtitutions = array(
       '&&' => '', //过滤了'&&'  
       ';' => '', //过滤了';'  
   );
   // Step3 将接收到的输入中与过滤列表匹配的字符过滤  
   $target = str_replace(array_keys($substitutions),$substitutions,$target);
   // Step4 根据操作系统的区别，对接收ip执行命令不同的ping操作
   if(stristr(php_uname('s'),'Windows NT')){
       //Windows
       $cmd = shell_exec('ping'.$target);
   }
   else{
       //*nix
       $cmd = shell_exec('ping -c 4'.$target);
   }
   // Final 过滤列表之过滤了"&&"和";"，可以使用其他命令字符注入  
   $html .= "<pre>{$cmd}</pre>";
}
?>

3.High_level

3.1 尝试注入

在ip地址后输入|ipconfig：

3.2 源代码

<?php 
if (isset($_POST['Submit'])){
   // Step1 代码接收用户输入的ip  
   $target = $_REQUEST['ip'];
   // Step2 设置过滤列表  
   $subtitutions = array(
       '&' => '',
       '&&' => '',   
       ';' => '', 
       '| ' => '', //仔细看这里过滤的是'| ',意味着'|'仍可利用
       '-' => '',
       '$' => '',
       '(' => '',
       ')' => '',
       '`' => '',
       '||' => '',
   );
   // Step3 将接收到的输入中与过滤列表匹配的字符过滤  
   $target = str_replace(array_keys($substitutions),$substitutions,$target);
   // Step4 根据操作系统的区别，对接收ip执行命令不同的ping操作
   if(stristr(php_uname('s'),'Windows NT')){
       //Windows
       $cmd = shell_exec('ping'.$target);
   }
   else{
       //*nix
       $cmd = shell_exec('ping -c 4'.$target);
   }
   // Final High等级的过滤列表比Medium增加了字符    
   $html .= "<pre>{$cmd}</pre>";
?>

4.Impossible_level

4.1 源代码

<?php
if(isset($_POST['Submit'])){
   checkToken($_REQUEST['user_token'],$_SESSION['session_token'],'index.php');//加入了Anti-CSRF Token
   $target = $_REQUEST['ip'];
   $target = stripslashes($target); //该函数会删除字符串string中的反斜杠，返回已剥离反斜杠的字符串。  
   $octet = explode(".",$target); //该函数把字符串打散为数组，返回字符串的数组。参数separator规定在哪里分割字符串，参数string是要分割的字符串，可选参数limit规定所返回的数组元素的数目。  
   if((is_numeric($octet[0]))&&(is_numeric($octet[1]))&&(is_numeric($octet[2]))&&(is_numeric($octet[3]))&&(sizeof($octet)==4)){ //检测string是否为数字或数字字符串，如果是返回TRUE，否则返回FALSE。  
       $target = $octet[0].'.'.$octet[1].'.'.$octet[2].'.'.$octet[3];//只有[int.int.int.int]格式的输入才执行
       if(stristr(php_uname('s'),'Windows NT')){
           $cmd = shell_exec('ping'.$target);
       }
       else{
           $cmd = shell_exec('ping -c 4'.$target);
       }
       $html .= "<pre>{$cmd}</pre>";
   }
   else{
       $html .= '<pre>ERROR:You have entered an incalid IP.</pre>';
   }
   
}
generateSessionToken();
?>
